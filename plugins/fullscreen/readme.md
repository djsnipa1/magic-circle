# @CreativeControls/Fullscreen
Enables the app to run in fullscreen

## Settings
```js
module.exports = {
  fullscreen: {

    // button triggers fullscreen of [editor/window]
    button: 'editor',

    // full screen on startup
    startup: false,

    // enables full screen for frame
    frame: true,

    // enables full screen for editor
    editor: true,
  }
}
```
